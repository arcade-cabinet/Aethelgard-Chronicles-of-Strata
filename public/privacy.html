Privacy Policy

Last updated: 2026-05-23

Aethelgard: Chronicles of Strata is an offline single-player game. This privacy policy documents what data the game touches, where it stays, and what it never does.

TL;DR. The game stores save data on your device. It does not send any data over the network. It does not track, profile, identify, or share you.

Data the game stores on your device

  • Save games. Stored in a local SQLite database (Capacitor on Android; sql.js + IndexedDB on web). Contains only the game state you chose to save.
  • Preferences. Audio mute, onboarding-seen flag, deterministic event-PRNG seed, unlocked achievements. Stored via Capacitor Preferences (native on Android, localStorage on web).

Data the game does NOT send anywhere

The game has zero outbound network requests in normal play. The Content-Security-Policy enforces connect-src 'self' blob:; every domain other than the app's own is rejected by the browser.

Third-party assets

The game bundles CC0 / royalty-free art and audio (KayKit, Kenney, PixelLoops, GameLoops). See the in-game Credits screen for the full list. These packs are static asset files — no runtime tracking or licence-server calls.

Children

The game is rated for all ages and has no chat, no UGC, no account system, no purchases, and no advertising.
