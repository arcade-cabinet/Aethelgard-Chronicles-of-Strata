# v0.12 Upgrade Graph

## Scope

Cycle goal (per PRD-v0.12 §2): **100+ upgrades** total, split as:

| Surface | Count today | Target v0.12 | Mechanic |
|---|---|---|---|
| In-match Discoveries | 36 (incl. 6 formations) | **76** (30 + 6 formations + 40 new) | gold-purchased mid-match; effect ends at match |
| Atelier meta unlocks | 30 | **42** (30 + 12 chain-starters) | lore-token-purchased cross-match |
| Per-chapter narrative perks | 0 | **12** (4 per chapter × 3 chapters) | granted by chapter at match start |
| **TOTAL** | **66** | **130** | |

That overshoots the 100+ target with headroom for cuts during
implementation.

## Why these counts

Each chain needs enough depth that mid-match decision-making
matters across a full ~20-minute RTS match. Player should hit
~6-8 Discoveries in a typical match; with 76 total they always
have meaningful choice. Atelier chain-starters give returning
players a chosen head-start without trivializing day-one matches
(starter purchases skip tier I only). Chapter perks signal each
campaign chapter's theme without bloating Atelier.

## Chain structure

Every in-match Discovery has:

- `id` — stable kebab-case slug
- `chain` — one of: `economy` | `military` | `diplomacy` | `magic` | `engineering` | `lore`
- `tier` — 1..4 (tier 1 = no prereqs; tier 4 = capstone)
- `specialisation` — chain-specific (see per-chain tables below)
- `cost` — `{ wood?, stone?, gold?, food? }`
- `prereqs` — array of upgrade ids (graph edges)
- `effect` — typed runtime effect (see "Effect kinds" below)

## Effect kinds

The v0.11 registry has 3 effect kinds: `multiply-harvest`,
`buff-combatant`, `flag`. v0.12 adds:

| Kind | Args | Surface |
|---|---|---|
| `multiply-harvest` | `{ factor }` | harvest tick rate |
| `buff-combatant` | `{ stat, delta, filter? }` | combat stat delta on every (matching) combatant |
| `flag` | `{ id }` | sets `game.research.flags[id]` for ad-hoc gates |
| **`buff-building`** | `{ buildingType, stat, delta }` | per-building stat (HP / DPS / output rate) |
| **`unlock-unit`** | `{ unitType }` | adds to `trainsUnits` for the gating building |
| **`unlock-building`** | `{ buildingType }` | makes the building constructible |
| **`unlock-formation`** | `{ formationId }` | already-purchased formation flag (today: flag-kind) |
| **`modify-cost`** | `{ resource, delta }` | scales economy costs for a class |
| **`modify-supply`** | `{ delta }` | global supply cap delta |
| **`reveal-tier`** | `{ tier }` | full-map vision tier (lore chain) |
| **`grant-resource`** | `{ resource, amount }` | one-shot grant (chapter perks) |

**Substance:** today the Economy chain is 5 linear entries. v0.12

## Chapter perks (12 new entries — 4 per chapter × 3 chapters)

Each campaign chapter grants 4 pre-purchased Discoveries at
match start, signaling its theme. These are NOT player choices —
they're chapter-defined.

### Chapter I (first-strata) — economy + survival theme

- `steel-plows` (Economy harvest I)
- `bulk-baskets` (Economy cap I)
- `forged-blades` (Military infantry I)
- `reinforced-walls` (Engineering defense I)

### Chapter II (pact-of-kings) — diplomacy theme

- `first-contact` (Diplomacy relations I)
- `exchange-policy` (Diplomacy trade I)
- `diplomatic-corps` (Diplomacy relations II)
- `iron-tipped-arrows` (Military archer I)

### Chapter III (necropolis-rising) — magic + arcane theme

- `mage-tower-aura` (Magic offense I)
- `crystalline-foci` (Magic offense II)
- `armored-saddles` (Military cavalry I — Knights vs undead)
- `cartography` (Lore reveal I — fog of war key)
